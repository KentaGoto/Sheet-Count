# Sheet-Count
Output the number of sheets and sheet names in the Excel file.  

## USAGE  
```pwsh  
$ Run.bat
> Dir: <DIR>
```
or  
```cmd  
$ Sheet-Count.py <FILE>
```
or
```cmd  
$ Sheet-Count.py
> File: <FILE>
```

## REQUIRES  
- Python 3  
- PowerShell  
- Batchfile  
- Windows
