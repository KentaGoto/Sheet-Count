# Sheet-Count
Excel sheet count.  

## USAGE  
```PowerShell  
$ .\Recursive-exect.ps1 -FolderPath <DIR>
```
or  
```PowerShell  
$ Sheet-Count.py <FILE>
```
or
```PowerShell  
$ Sheet-Count.py
> File: <FILE>
```

## REQUIRES  
- Python 3  
- PowerShell  
- Windows
